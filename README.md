## 🧠 Project Overview

The **WhatsApp Chat Analyzer** is an interactive data analytics dashboard that analyzes exported WhatsApp chats and generates meaningful insights such as:

✅ Message statistics  
✅ User activity analysis  
✅ Emoji usage trends  
✅ Word frequency analysis  
✅ Activity heatmap  
✅ Timeline visualization  

This project demonstrates **real-world data analysis workflow** used by professional Data Analysts.
